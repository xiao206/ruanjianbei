# AI 智能匹配与能力图谱系统测试报告

## 1. 测试目的

本测试报告旨在验证 AI 智能匹配与能力图谱系统的功能完整性、性能稳定性和用户体验，确保系统能够在银河麒麟操作系统和 LoongArch 架构上正常运行，满足业务需求。

## 2. 测试环境

### 2.1 硬件环境
- **CPU**: 4核 LoongArch 处理器
- **内存**: 8GB
- **磁盘**: 256GB
- **网络**: 100Mbps 局域网

### 2.2 软件环境
- **操作系统**: 银河麒麟高级服务器操作系统 V11 (LoongArch)
- **JDK**: 华为毕升 JDK 17
- **数据库**: 达梦DM8
- **图数据库**: Neo4j Community 5.19.0
- **向量数据库**: Milvus 2.4.0
- **AI 服务**: Ollama + Qwen2.5-3B
- **前端服务**: Nginx 1.20.0
- **浏览器**: Chrome 90+
