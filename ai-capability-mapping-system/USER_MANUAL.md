# AI 智能匹配与能力图谱系统用户手册

## 1. 系统概述

AI 智能匹配与能力图谱系统是基于银河麒麟操作系统的智能化能力评估与匹配平台，利用人工智能技术实现技术人员能力与需求的高效精准适配。

### 1.1 核心功能

- **文档智能解析**：支持 PDF、Word、图片等多格式文件上传与解析
- **双向能力图谱**：构建人才能力图谱和需求图谱，支持多种可视化布局
- **供需智能匹配**：基于向量检索和知识图谱推理实现智能匹配
- **后台管理**：用户管理、技能库管理、系统监控

### 1.2 系统架构

- **前端**：Vue 3 + Element Plus + AntV G6
- **后端**：Spring Boot 微服务
- **数据库**：MySQL / 达梦 DM8、Neo4j、Milvus、Redis
- **AI 服务**：Ollama、PaddleOCR、BGE-M3

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问系统地址：http://localhost:3000
2. 点击右上角 "登录" 按钮
3. 输入用户名和密码：
   - 管理员：admin / admin123
   - 普通用户：user1 / user123
4. 点击 "登录" 按钮进入系统

### 2.2 系统界面

系统主界面包含以下模块：
- **首页**：系统概览和功能入口
- **文档解析**：文件上传和解析
- **能力图谱**：图谱可视化和交互
- **智能匹配**：人才与需求匹配
- **后台管理**：系统管理功能（仅管理员可见）

## 3. 文档智能解析

### 3.1 文件上传

1. 点击左侧菜单 "文档解析"
2. 在上传区域拖拽文件或点击 "点击上传"
3. 选择要上传的文件（支持 PDF、Word、图片等格式）
4. 系统自动开始解析过程

### 3.2 解析状态查看

1. 上传完成后，系统会显示文件信息和解析状态
2. 点击 "检查状态" 按钮查看解析进度
3. 解析完成后，点击 "查看结果" 按钮查看解析结果

### 3.3 解析结果查看

解析结果包含：
- **技能信息**：从文档中提取的技能和等级
- **项目经验**：从文档中提取的项目经验
- **其他信息**：教育背景、工作经历等

## 4. 能力图谱

### 4.1 图谱类型选择

1. 点击左侧菜单 "能力图谱"
2. 在顶部选择图谱类型：
   - 人才能力图谱：展示人才的能力结构
   - 需求图谱：展示需求的技能要求

### 4.2 图谱加载

1. 输入实体 ID（如人才 ID 或需求 ID）
2. 点击 "加载图谱" 按钮
3. 系统会加载并显示相应的能力图谱

### 4.3 图谱操作

- **放大/缩小**：使用工具栏的放大/缩小按钮
- **重置布局**：点击 "重置布局" 按钮
- **切换布局**：点击 "切换布局" 按钮在力导向布局和层次布局之间切换
- **拖拽**：拖动节点调整位置
- **悬停**：鼠标悬停在节点上查看详细信息

## 5. 智能匹配

### 5.1 执行匹配

1. 点击左侧菜单 "智能匹配"
2. 输入需求 ID
3. 点击 "执行匹配" 按钮
4. 系统会执行匹配过程并显示匹配结果

### 5.2 匹配结果查看

匹配结果包含：
- **人才姓名**：匹配的人才名称
- **匹配度**：匹配程度的可视化展示
- **匹配分数**：具体的匹配分数
- **匹配技能**：匹配的技能列表
- **技能差距**：未匹配的技能列表

### 5.3 匹配详情

1. 点击 "查看详情" 按钮查看匹配详情
2. 详情页面包含：
   - 技能匹配：详细的技能匹配情况
   - 学习路径：基于技能差距的学习建议

## 6. 后台管理

### 6.1 用户管理

1. 点击左侧菜单 "后台管理"
2. 选择 "用户管理" 标签
3. 可以查看、添加、编辑、删除用户
4. 可以设置用户角色和权限

### 6.2 技能管理

1. 点击 "技能管理" 标签
2. 可以查看、添加、编辑、删除技能
3. 可以设置技能分类和等级

### 6.3 系统监控

1. 点击 "系统监控" 标签
2. 可以查看系统各服务的运行状态
3. 可以查看数据库和 AI 模型的状态

## 7. 系统设置

### 7.1 个人设置

1. 点击右上角用户头像
2. 选择 "个人设置"
3. 可以修改个人信息和密码

### 7.2 系统配置

## 8. 常见问题

### 8.1 登录问题

- **忘记密码**：联系管理员重置密码
- **登录失败**：检查用户名和密码是否正确
- **账号被锁定**：联系管理员解锁

### 8.2 文档解析问题

- **解析失败**：检查文件格式是否支持
- **解析速度慢**：较大的文件可能需要更长时间
- **解析结果不准确**：尝试重新上传或手动编辑解析结果

### 8.3 图谱问题

- **图谱加载失败**：检查网络连接和服务状态
- **图谱显示异常**：尝试刷新页面或重置布局
- **节点过多**：系统会自动优化显示，也可以使用搜索功能定位特定节点

### 8.4 匹配问题

- **匹配结果不准确**：检查需求描述是否清晰
- **匹配速度慢**：复杂的匹配可能需要更长时间
- **无匹配结果**：可能需要调整匹配参数或扩展技能库

### 11.1 系统故障

- **服务不可用**：检查服务状态和日志

### 11.2 数据问题

- **数据丢失**：检查备份和恢复机制
- **数据不一致**：检查数据同步和一致性机制
- **数据安全**：检查数据加密和访问控制

### 11.3 性能问题

- **响应缓慢**：检查系统资源和负载
- **内存不足**：增加系统内存或优化内存使用
- **CPU 使用率高**：优化代码或增加 CPU 资源

## 15. 术语解释

### 15.1 核心概念

- **能力图谱**：展示人才能力结构的可视化图表
- **需求图谱**：展示需求技能要求的可视化图表
- **向量检索**：基于语义相似度的检索方法
- **知识图谱推理**：基于知识图谱的关系推理
- **LLM**：大语言模型，用于文本理解和信息抽取
- **OCR**：光学字符识别，用于识别图片中的文字

### 15.2 技术术语

- **Spring Boot**：Java 微服务框架
- **Vue 3**：前端 JavaScript 框架
- **Neo4j**：图数据库
- **Milvus**：向量数据库
- **Ollama**：本地 LLM 推理引擎
- **PaddleOCR**：OCR 引擎
- **BGE-M3**：文本 embedding 模型

### 15.3 业务术语

- **技能等级**：技能掌握程度的评估标准（了解、掌握、精通、专家、权威）
- **匹配度**：人才与需求的匹配程度
- **技能差距**：人才与需求之间的技能差异
- **学习路径**：基于技能差距的能力提升建议
- **RBAC**：基于角色的访问控制
